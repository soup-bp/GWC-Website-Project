The three most important elements needed for a responsive, sleek website (like this one :P) is an HTML file for the core content, CSS stylesheet to make it look pretty, and JavaScript code to make the pretty stuff even fancier (more responsive to clicks and user input and stuff).

STEP 1:

Have an Idea

This is arguably the most important step. If you don't have an idea to execute, don't bother wasting your time building a full-fledged website. Sure, it's good to practice and sharpen your skills in HTML/CSS/JavaScript every once in a while, but if you don't have a concrete, realistic idea to execute don't spend too much time working on creating a website for it.

STEP 2:

Make Your HTML File and CSS Stylesheet

  1. Being organized is very attractive and cool (Source: urmom.edu)
  2. I save everything to my desktop and it is actual chaos whenever I look for something so please for the love of all things holy make folders AND ACTUALLY USE THEM PLEASE
